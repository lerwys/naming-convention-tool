#{msgs.title} API Manual

Data in the Naming Service can be accessed programmatically through a RESTful Application Programming Interface (API). The Naming System API follows Semantic Versioning for its version numbers. The current API version number is #{msgs.apiversion}.

Representations

The representations of the resources must be specified through HTTP ACCEPT header. This API supports JSON and XML representations. Examples:
  • Accept: application/json
  • Accept: application/xml

Base URL

The base URL for Resource Identifiers is #{restRoot}.
